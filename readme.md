# 동양미래 데이버분석 리포트 저장소

매 주차별 학습한 내용을 깃 `풀-리퀘스트`로 제출합니다. 
1. 저장소를 포크한다.
2. 자신의 브랜치를 만든다. (브랜치 안에 자기 이름의 마크다운 파일을 만들어 주세요)
3. 매 주차별 폴더를 만들고 실습한 마크다운 파일과 확인 이미지를 첨부한다.
4. 풀 리퀘스트를 전송한다. (제목은 : XX주차 000 )
